# 🛡️ Investalo ScamScreener: Don't get rekt! 🤖

Hand aufs Herz: Der Finanzmarkt ist voll von Scams. Mit **Invesalo** baue ich Tools, die dem Nutzer den Rücken freihalten. Der **ScamScreener** ist mein erster Bodyguard gegen manipulative Nachrichten und "Schnell-reich-werden"-Müll.

## 🧠 Warum dieser Ansatz?
Ich wollte nicht nur eine einfache KI-Abfrage. Deshalb habe ich ein **hybrides System** gebaut:
1. **OpenAI (The Brain):** Versteht den Kontext, die Psychologie hinter dem Scam und die fiesen Tricks.
2. **Local ML (The Guard):** Ein Random Forest Modell, das direkt auf meinem Code läuft. Es lernt aus jedem Fall dazu und erkennt Muster, die man mit bloßem Auge übersieht.

## 🔥 Tech-Highlights
* **FastAPI Backend:** Schlank, schnell und asynchron.
* **Hybrid Scoring:** 70% KI-Expertise + 30% lokales Machine Learning.
* **Self-Improving:** Dank des `train.py` Skripts lernt der Screener mit jeder neuen Scam-Welle dazu.
* **Modern UI:** Ein sauberes Interface, das Vertrauen schafft.
